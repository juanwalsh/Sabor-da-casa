6. SISTEMA DE PONTOS - COMO FUNCIONA
================================================================================

- Cliente identificado por NOME + TELEFONE
- Cada R$ 1,00 gasto = 1 ponto
- Pontos acumulam em compras futuras
- API /api/customers gerencia clientes e pontos
- Ao fazer pedido, sistema automaticamente:
  * Cria cliente se nao existe
  * Atualiza pontos se ja existe
  * Registra total gasto e numero de pedidos

================================================================================
7. ESTRUTURA DA API
================================================================================

POST /api/products     - Criar produto
GET  /api/products     - Listar produtos
PUT  /api/products     - Atualizar produto
DELETE /api/products?id=xxx - Remover produto

POST /api/orders       - Criar pedido
GET  /api/orders       - Listar pedidos
PUT  /api/orders       - Atualizar status do pedido

POST /api/customers    - Criar/atualizar cliente e pontos
GET  /api/customers    - Listar clientes
GET  /api/customers?phone=xxx&name=xxx - Buscar cliente especifico
PUT  /api/customers    - Atualizar cliente / resgatar pontos

GET  /api/init         - Inicializar dados (se nao existirem)
POST /api/init         - Forcar reinicializacao dos dados
